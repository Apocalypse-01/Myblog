# 🎨 首页三栏布局界面生成提示词

## 📸 参考图片描述

**原始图片特征分析：**

### 整体布局结构

- **背景层**：全屏动漫风格背景图（二次元/动漫人物），覆盖整个视口
- **前景层**：三个半透明矩形面板，呈"左-中-右"三栏排列
- **间距系统**：
  - 上下边距：约60px（顶部和底部留白）
  - 左右边距：约80px（左右两侧留白）
  - 面板间距：约30px（左右面板与中间面板的间距）
  - 面板圆角：12px（所有面板统一）

### 三栏详细规格

#### 🔲 左侧栏（Left Panel）

- **位置**：页面左侧，固定宽度
- **尺寸**：宽度280px × 高度70vh
- **颜色**：深灰褐色半透明 `rgba(40, 35, 45, 0.65)`
- **效果**：
  - 毛玻璃模糊：`backdrop-filter: blur(12px)`
  - 边框：`1px solid rgba(255, 255, 255, 0.1)` （极淡白色边框）
  - 圆角：`border-radius: 12px`
- **视觉特点**：最暗的区域，接近黑色但不纯黑

#### ⬜ 中间主内容区（Center Panel）

- **位置**：页面正中央，自适应宽度
- **尺寸**：最大宽度600px × 高度75vh（比两侧稍高稍宽）
- **颜色**：深蓝灰色半透明 `rgba(30, 30, 40, 0.7)`
- **效果**：
  - 毛玻璃模糊：`backdrop-filter: blur(15px)` （模糊度略强于两侧）
  - 边框：`1px solid rgba(255, 255, 255, 0.15)` （边框略明显）
  - 圆角：`border-radius: 12px`
- **视觉特点**：
  - 透明度最低（0.7），最不透明
  - 高度最高（75vh），视觉重心
  - 宽度弹性（flex: 1），占据剩余空间

#### 🔳 右侧栏（Right Panel）

- **位置**：页面右侧，固定宽度
- **尺寸**：宽度280px × 高度70vh
- **颜色**：深红棕色/暗红色半透明 `rgba(120, 50, 40, 0.65)`
- **效果**：
  - 毛玻璃模糊：`backdrop-filter: blur(12px)`
  - 边框：`1px solid rgba(255, 200, 180, 0.2)` （暖色调边框）
  - 圆角：`border-radius: 12px`
- **视觉特点**：
  - 唯一暖色调区域（红棕色调）
  - 与左侧形成冷暖对比
  - 边框带有微弱的橙红色调

## 🎯 核心设计原则

### 1️⃣ 视觉层次（Visual Hierarchy）

```
层级1（最深）：中间主内容区 → opacity: 0.7, blur: 15px
层级2（中等）：左侧栏       → opacity: 0.65, blur: 12px  
层级3（中等）：右侧栏       → opacity: 0.65, blur: 12px
层级4（底层）：全屏背景图   → 100%不透明，无模糊
```

**目的**：引导视线聚焦中间区域

### 2️⃣ 色彩心理学应用

```
左侧（冷色调）：深灰褐 → 稳重、专业、辅助信息
中间（中性色）：深蓝灰 → 核心、聚焦、主要内容
右侧（暖色调）：深红棕 → 温暖、活力、互动功能
```

**目的**：通过色彩暗示功能区划分

### 3️⃣ 玻璃拟态（Glassmorphism）风格

```
核心要素：
✅ 半透明背景（opacity: 0.65-0.7）
✅ 背景模糊（blur: 12-15px）
✅ 微妙边框（rgba白色，10%-20%透明度）
✅ 柔和阴影（可选）
✅ 圆角处理（12px）
```

**目的**：现代感、通透感、层次感

## 📐 技术实现规范

### HTML结构要求

```html
<div class="home-container">
    <!-- 全屏背景（伪元素或独立div） -->
    
    <div class="left-panel">
        <!-- 左侧内容 -->
    </div>
    
    <div class="center-panel">
        <!-- 主内容区 -->
    </div>
    
    <div class="right-panel">
        <!-- 右侧内容 -->
    </div>
</div>
```

### CSS关键属性清单

#### 容器（Container）

```css
.home-container {
    display: flex;              /* 弹性布局 */
    justify-content: center;   /* 水平居中 */
    align-items: center;       /* 垂直居中 */
    gap: 30px;                 /* 面板间距 */
    padding: 60px 80px;        /* 内边距（上下60，左右80）*/
    width: 100vw;              /* 视口宽度 */
    height: 100vh;             /* 视口高度 */
}
```

#### 面板通用样式

```css
.panel {
    border-radius: 12px;       /* 统一圆角 */
    backdrop-filter: blur();   /* 毛玻璃效果 */
    border: 1px solid rgba();  /* 半透明边框 */
    /* 高度使用vh单位，响应式 */
}
```

#### 响应式断点

```
桌面端（>1200px）：
  • 保持三栏横向布局
  • 左右栏固定280px
  • 中间栏max-width: 600px
  
平板端（900px-1200px）：
  • 缩小间距至20px
  • 左右栏缩小至220px
  • 中间栏max-width: 500px
  
移动端（<900px）：
  • 切换为纵向单列布局（flex-direction: column）
  • 所有面板width: 100%
  • 面板高度改为auto或min-height: 300px
  • 间距调整为15px
```

### 效果C：渐变遮罩

```css
/* 面板内上下渐变，增加深度感 */
.panel {
    background: linear-gradient(
        to bottom,
        rgba(30, 30, 40, 0.8) 0%,
        rgba(30, 30, 40, 0.6) 50%,
        rgba(30, 30, 40, 0.8) 100%
    );
}
```

```css

    from {
        opacity: 0;
        transform: translateY(30px);
    }
    to {
        opacity: 1;
        transform: translateY(0);
    }

.center-panel { animation: fadeInUp 0.6s ease 0.2s both; }
.right-panel { animation: fadeInUp 0.6s ease 0.3s both; }
```

## ⚠️ 注意事项

### 性能优化提醒

1. **backdrop-filter性能消耗大**
   - 移动端慎用或降低模糊半径
   - 考虑降级方案（纯色背景替代）
2. **背景图优化**
   - 格式：WebP > JPEG > PNG
   - 尺寸：不超过1920×1080
   - 压缩：质量80%，文件<500KB
   - 懒加载：非首屏可延迟加载
3. **响应式测试必做项**
   - iPhone SE (375px)
   - iPad (768px)
   - MacBook Air (1280px)
   - Desktop (1920px)

### 浏览器兼容性

- ✅ Chrome/Edge 76+
- ✅ Firefox 103+
- ✅ Safari 9+ (iOS)
- ⚠️ IE不支持 backdrop-filter（需polyfill）
