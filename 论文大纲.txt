二、摘要

2.1 中文摘要
本论文设计并实现了一个基于Spring Boot框架的员工信息管理系统。研究针对传统员工信息管理方式存在的流程繁琐、数据整合困难、权限管控模糊等问题，采用Java技术和MySQL数据库，利用Spring Boot+Vue前后端分离架构，构建了集高效、智能、安全于一体的管理系统。系统实现了管理员管理、公告管理、薪资管理、员工考勤管理、员工请假管理、员工管理等核心功能模块，为企业人力资源管理提供了信息化解决方案。

2.2 英文摘要
Abstract: This paper designs and implements an employee information management system based on Spring Boot framework. Addressing issues in traditional employee information management such as cumbersome processes, difficult data integration, and ambiguous authority control, the system adopts Java technology and MySQL database, utilizing Spring Boot + Vue front-end and back-end separation architecture to build a management system integrating high efficiency, intelligence, and security. The system implements core functional modules including administrator management, announcement management, salary management, employee attendance management, employee leave management, and employee management, providing an information-based solution for enterprise human resource management.

2.3 关键词
中文关键词：信息管理、高效、Spring Boot、MySQL数据库
英文关键词：Information management, High efficiency, Spring Boot, MySQL database
